## Description
F1-Universe is a modern, responsive web application dedicated to the upcoming Formula 1 2026 season. It features a race calendar with interactive maps, a drivers showcase (including new entries like Audi), and a contact form. Built with React and Vite, focusing on clean code and component-based architecture.

## Third-party Components
- **Leaflet & React-Leaflet**: Used for displaying the interactive map of the next Grand Prix location.
- **React Router DOM**: Handles client-side routing between pages.

## Installation
To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/F1-Universe.git
   ```
2. Navigate to the project directory:
   ```bash
   cd F1-Universe
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
